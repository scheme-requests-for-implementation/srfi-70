This repository hosts [[http://srfi.schemers.org/srfi-70/][SRFI 70]]: Numbers, a [[http://srfi.schemers.org/][Scheme Request for Implementation]].

If you'd like to participate in the discussion of this SRFI, or report issues with it, please [[http://srfi.schemers.org/srfi-70/][join the SRFI-70 mailing list]] and send your message there.
